# Design Tokens

Design tokens are the visual design atoms of the **Print Room Studio** design system — named entities that store visual attributes. They live in `packages/ui/tailwind.config.ts` and `packages/ui/src/styles/globals.css`, mirrored in `apps/storybook/src/tokens/design-tokens.json` for documentation.

> **Editing Tokens:** Open `design-tokens.json` in any text editor (or GitHub's web editor), change the value, and commit. Storybook rebuilds automatically. See the **Design Tokens Guide** page for full instructions.
