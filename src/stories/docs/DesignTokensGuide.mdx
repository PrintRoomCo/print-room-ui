# Design Tokens Guide

> **For Non-Developers:** This guide explains how to update brand colours, typography and spacing without writing code. For the full visual reference of every token, see the **[Design Tokens](?path=/docs/documentation-design-tokens--docs)** page.

### Example Edit

```json
"colors": {
  "brand": {
    "primary": "#2B3990",       // ← Print Room Blue
    "primaryDark": "#232E73",   // ← Darker variant
    "primarySoft": "#3A4BA8",   // ← Lighter variant
    "accent": "#0900FF",        // ← Figma accent blue
    "accentLime": "#F1FFA5"     // ← Lime highlight
  }
}
```

## Token File Structure

The JSON file is organised into these top-level sections:

<div style={{ display: 'grid', gridTemplateColumns: 'repeat(auto-fill, minmax(200px, 1fr))', gap: '12px', padding: '16px 0' }}>
  {[
    { key: 'colors.brand', desc: 'Primary brand hex values' },
    { key: 'colors.figma', desc: 'Legacy Figma design colours' },
    { key: 'colors.ui', desc: 'Light-mode CSS variables' },
    { key: 'colors.uiDark', desc: 'Dark-mode CSS variables' },
    { key: 'colors.palette', desc: '50–950 colour scales' },
    { key: 'colors.status', desc: 'Success / Warning / Error / Info' },
    { key: 'colors.chart', desc: 'Data-viz colour series' },
    { key: 'typography', desc: 'Font families, sizes, weights' },
    { key: 'spacing', desc: '4 px base-unit scale' },
    { key: 'borderRadius', desc: 'Radius tokens + CSS var' },
    { key: 'shadows', desc: 'Elevation + glass morphism' },
    { key: 'animation', desc: 'Keyframes & durations' },
  ].map((s) => (
    <div key={s.key} style={{ background: '#f8fafc', border: '1px solid #e2e8f0', borderRadius: '8px', padding: '12px' }}>
      <span style={{ fontSize: '13px', fontWeight: 600, color: '#0f172a', fontFamily: "'JetBrains Mono', monospace" }}>{s.key}</span>
      <p style={{ margin: '4px 0 0', fontSize: '12px', color: '#64748b' }}>{s.desc}</p>
    </div>
  ))}
</div>

## Need Help?

If you're unsure about a change, create a **Pull Request** instead of committing directly. This lets the team review before it goes live.

For the complete visual reference of every colour, typography specimen, spacing scale, shadow elevation and animation, see the **[Design Tokens](?path=/docs/documentation-design-tokens--docs)** page.
